# Installation 
To setup the app for development on your local device, follow the following steps:
1. Clone the repo 
````
git clone https://github.com/akash-10-23/covid-19_tracker.git
cd covid-19_tracker
````
3. Install all the packages
````
npm i
````
5. Run the development server
````
npm start
````
